# 🚀 KONTAKT — Smart Contact Manager

A modern, feature-rich **Contact Management Application** built using Python, designed with a clean UI and real-world usability in mind.

---

## ✨ Key Features

### 🧠 Smart Functionality

* 🔤 **Auto Avatar Generator**
  Automatically creates profile images using the first letter of the name (similar to Gmail)

* 🧾 **Duplicate Detection**
  Prevents duplicate contacts using phone/email validation

* 📊 **Sorting System**
  Organize contacts by:

  * Newest First
  * A → Z
  * Z → A

---

### ⚙️ Core Features

* ➕ Add Contacts
* ✏️ Update Contacts
* ❌ Delete Contacts
* 🔍 Search (Name / Phone / Email)
* 🏷️ Tag-based Filtering (Work, Family, Friend, etc.)
* 📤 Export Contacts to CSV

## ⚡ Getting Started

### 1. Clone the repository

```
git clone https://github.com/sanskaaar01/kontakt-manager.git
cd kontakt-manager
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
python app.py
```
